F.A.Q. for the Pixelmon Server

How do I connect to the server?

  1. Download the Technic Launcher: https://www.technicpack.net/download
  2. Log in with your Mojang account
  3. Install the Pixelmon Reforged Pack
  4. Connect to our normal server address (16colorgames.com:25565)! You will be automatically redirected to the Pixelmon server

Minecrft keeps crashing when I try to connect!

You probably have the 32-bit version of Java installed. Go here and get the 64-bit version for your system. You will need to make an Oracle account.

Whats up with the hunts?

There are several pokemon hunts running at any given time. Using /hunt you can view the current ones. If you catch a pokemon currently being hunted and it has the specified nature, you will get several rare candies, some pokeballs, and occasionally some other useful items!

I want to help out and get free stuff!

Gyms

Natural gyms are spawning a large distance apart. If you want to help build big structures, you can help out here! Check out the map to see the distance between the Aubrey Gym and the (SECOND TOWN VOTE IN PROGRESS) Gym; this is roughly how far apart we would like the gyms to be. Build up a Gym themed around a specific type with room for at least 5 trainers, and contact oa10712 to recieve a random shiney of the type of gym you built!

Themed Areas

Unlike in the games, there are very few "man-made" structures with specific spawns. You can help out by building these. Some examples are the Power Plants, Sewers, and Ghost Towers from various games. These can be built in towns or preferably along routes. The bigger, the better with these structures, as we can create custom spawning rules for these structures. Contact oa10712 after building the structure to recieve a shiney from the spawnset of the new building!
